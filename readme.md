# Tetris

A lightweight wrapper for the Tetris website - written in Kotlin and Jetpack Compose. Requires internet access on first launch in order to cache the webview.

This project is heavily inspired by [Halkiion's Implementation](https://github.com/Halkiion/TetrisWebView-Android), this mostly exists because it had a couple bugs and I didn't want to write Java to fix them.

## Installation

1. Go to the [Release](https://github.com/Importantamigo/TetrisWebView-kt/releases) section.
2. Download the apk corresponding to your device's architecture, if you're not sure just download [app-universal-release.apk](https://github.com/Importantamigo/TetrisWebView-kt/releases/download/release-v1.0.0/app-universal-release.apk).
3. Install the app once the download is done. Ignore any pl
4. Enjoy :)

## Troubleshooting

- App is bulletproof, if any problem arises just close and restart with internet enabled.
- Still have problems? Open an issue with a bug report or feature request [here](https://github.com/Importantamigo/TetrisWebView-kt/issues).
